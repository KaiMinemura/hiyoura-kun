# ひようら君

## 目的
日吉駅には50店舗以上の飲食店があり、その中からお店を選ぶのはたいへんです。  
そこで「迷わずお店を選ぶ」というコンセプトを基にして「ひようら君」を作りました。

## 説明
日吉駅の繁華街（通称：ひようら）にある、おすすめの飲食店を紹介してくれるLINE Bot
